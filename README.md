## Phandelver and Below: The Shattered Obelisk - Crypt of the Talhund Lock Puzzle

This is a representation of the Crypt of the Talhund puzzle.

I found this image on reddit, all credit on the image goes to [golden-phlox on reddit](https://www.reddit.com/user/golden-phlox/) who created the image of the lock and put it up [on DMs Guild](https://www.dmsguild.com/en/product/516620/unlocking-the-crypt-of-talhund-puzzle-the-shattered-obelisk) and [on reddit](https://www.reddit.com/r/LostMinesOfPhandelver/comments/1jj2sbl/puzzle_lock_for_talhundereth_the_crypt_of_the/).

I've just taken this and made it into a simple web app that allows a player to work out the puzzle in a web browser.
